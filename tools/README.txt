Creating an uncompressed cubemap .ktx file
$ tools/toktx.exe --cubemap skybox.ktx data/textures/skybox/right.ppm data/textures/skybox/left.ppm data/textures/skybox/top.ppm data/textures/skybox/bottom.ppm data/textures/skybox/back.ppm data/textures/skybox/front.ppm 

For creating .ktx files with compressed images use "PVRTexTool" from Imagination.
